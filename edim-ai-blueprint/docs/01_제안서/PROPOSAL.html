제안 개요
(Executive Summary)

본 제안서는 [고객사명]의 AI 기반 2D 도면 생성 시스템(edim-ai-blueprint) 구축 사업을 위한 솔루션 제안입니다. 본 시스템은 웹 브라우저에서 기존 CAD 파일(DXF·DWG·IFC)을 업로드하여 2D 도면으로 렌더링하고, AI 프롬프트(자연어)로 도면 초안을 자동 생성하며, 결과를 표준 DXF 파일로 내보낼 수 있는 경량 웹 CAD 시스템입니다. [회사명]은 정규화된 DrawingDocument JSON 스키마를 중심으로 한 확장 가능한 아키텍처와 Anthropic Claude API 기반 AI 도면 생성 기술을 통해, 고객의 레거시 CAD 자산 활용도 제고와 도면 작성 생산성 향상이라는 비즈니스 목표 달성을 지원합니다.

솔루션 구성
(Solution Architecture)

[프론트엔드] React(Vite+TypeScript) — SVG 2D 렌더러(팬/줌, 레이어 색상/토글, CAD Y-up 좌표 변환), 파일 업로드·AI 프롬프트·내보내기 패널
↕ REST API
[백엔드] Python FastAPI — DXF 임포터/익스포터(ezdxf, AutoCAD R2010), IFC 2D 평면 추출(ifcopenshell, storeyIndex), DWG→DXF 변환 연동(ODA File Converter, 플러그블), AI 도면 생성기(Anthropic Claude API, 키 미설정 시 스텁)
↕
[공통 데이터 모델] 모든 임포터와 AI 생성기가 하나의 정규화된 DrawingDocument JSON(line/polyline/circle/arc/text 엔티티 + 레이어)을 생성하고, 프론트엔드는 이 형식만 렌더링
