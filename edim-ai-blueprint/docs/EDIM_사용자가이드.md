# EDIM 사용자 가이드

> B22 산출물 (2026-07-10) — 모듈별 화면 조작 안내. 대상: 일반 사용자(GENERAL)·설계 Set-up(SETUP).
> 접속: https://edim.seekerslab.com — 사번/비밀번호 로그인 (5회 실패 시 자동 잠금 → 관리자 해제).

## 공통 조작

| 조작 | 방법 |
|---|---|
| 화면 열기 | 좌측 트리 클릭 (활성 화면은 트리에 하이라이트) — 상단 헤더에서 모듈(CPQ/PLM/Code/ERP/Toolbox/공통) 전환 |
| F-Key | F2 신규 · F3 삭제 · F8 조회 · F9 Run · F12 저장 — 상태바 셀 클릭도 동일 |
| 탭 | Alt+←/→ 이동 · Alt+W 닫기 · Alt+1~9 n번째 · 브라우저 뒤로/앞으로 = 탭 이력 |
| 검색 | Ctrl(⌘)+K — 화면·코드·문서·도면 통합 검색 |
| 실행 취소 | Ctrl+Z / Ctrl+Y — Design Editor 치수·데이터 Table 행·UI Designer 배치 |
| 상세 진입 | 그리드 더블클릭 = 상세 탭 (코드·문서·부품·이벤트) |
| 데이터 소스 | 상태바 우측 `DB: EDIM-PRD` = 실DB · `DB: MOCK` = 백엔드 불가(쓰기 거부됨) |
| 언어 | 타이틀바 KO/EN/JA/ZH — 즉시 전환 |

## CPQ

- **제품 선정 (C-1)**: 슬롯 콤보 변경 → BOM 실시간 재전개. `Spec Excel ⬆` 로 사양 일괄 입력. `Quote Preview` = 즉석 견적 PDF. F9 = EDIM Run 실행.
- **EDIM Run**: 6단계 파이프라인(BOM→치수→도면→원가→견적/BOM→저장) 진행 표시. 완료 후 산출물 그리드(미리보기=CAD 뷰어·⬇ 다운로드)와 **원가 패널** — `PCR 생성`(기여마진·EBIT) → `견적 확정`(QT 채번·PDF 미리보기).
- **문서함 (M-5-4)**: 상태 필터·Grade 통제. 문서 더블클릭 = 상세(실채번·미리보기/Print = 워터마크 렌더).

## PLM

- **Design Editor (S-4-1-1)**: 치수 더블클릭 = 편집, F9 = Macro 재평가 + CAD 재작도, F12 = 임시저장. 우측 **Simulation** 판넬 — VARIANT 값 변경 시 MACRO 치수 즉시 재계산(Δ 표시) → `적용`. `DXF 열기/내보내기`. 편집 모드 Block 더블클릭 = 부품 상세(조립순서 ◆ 실데이터).
- **도면 대장 (M-4-1)**: 등록 F2 → Rev 올리기 → Supersedure 대체. 상세 탭 [Rev 이력|승인 단계|Variants|Referencers|첨부] — 승인 단계는 작성→검토→승인 순서 강제, 반려 시 DRAFT 복귀. 행 더블클릭 = 연결 DXF CAD 뷰어(레이어 색·굵기 편집, 측정 M).
- **부품 대장 (M-4-7)**: 부품 등록 F2(재질·공급처·코드 연결) — 우측 공급자 코드 매핑(발주서 병기용). BOM 참조 중 부품은 삭제 보호.
- **Work Process (S-4-1-2)**: MAKE/BUY 토글 → F12 저장 (새로고침 유지).

## Code Management

- **Sub Code 등록 (S-1-1)**: 중복검토 → 승인 요청(PENDING).
- **Code Relationship (S-1-4)**: `＋ Add` 로 Child 실등록(DRAFT) → **Running Test 통과해야 승인 요청 가능** (CODE-009).
- **데이터 Table (M-3-7)**: Excel 식 셀 편집 · Excel Import · F3 행 삭제.
- **Hierarchy 주소 (M-3-1)**: 노드 등록 F2(선택 노드가 상위 — 주소 접두 검증)·개명·삭제 F3. 주소가 Macro/Table/Code 참조 경로.

## ERP

- **Project 등록 (S-3-5)**: 영업 단계 상태기계. `중복검토` = 실질의. `Print Set-up 호출` = S-3-4 직행.
- **발주 PR·PO (M-8-2)**: Stock Check F8(재고 충족 제외) → 품목 선택 → `견적 요청(QCR)` 발행 또는 `발주 생성` — 조건 다이얼로그(납품/운송/최소수량/인증서) 입력 후 확정하면 **발주서가 문서함에 등록**되고 공급자 코드가 병기됩니다.
- **창고·저장위치 (M-8-4)**: REGION→PLANT→WAREHOUSE→STORAGE→SECTOR 5계층 (순서 강제). 위험물 허용·검사주기 관리.
- **단가 관리 (M-12-5)**: ＋ 등록·Excel Import·resolve 시뮬레이션(적용→구매→재고→견적 우선순위). 재고 단가는 최고/최저/평균/최근 4종.
- **Dashboard (M-14-4)**: 공정 이상 경고 더블클릭 = 이벤트 상세(재배정·에스컬레이션).

## Toolbox

- **Macro Studio (S-2-2)**: Prompt → `▶ 생성 (AI)` → 수식 편집 → `Run F9`(Test) → TESTED 후 `검증·승인 요청`. **저장(4-Way)** = 수식+코드+플로차트+설명+Test 전부 영속·복원. 모드 콤보 MACRO/CODING(코드 기반, 엔진 v1 은 수식만 실행). `기능 찾기` — 자연어 검색(예: "합계") 후 클릭 = 수식 삽입. 참조 칩 = 이 Macro 가 쓰는 Table.
- **UI Designer (S-2-1)**: 팔레트 클릭 배치 → Object Inspector → `미리보기`(동적 렌더) → F12 저장 → 게시(승인).
- **Templet 관리 (S-2-3)**: JSON 정의 편집 → F12 저장(DRAFT 회귀).

## 공통 모듈

- **승인함 (M-15-2)**: 요청 클릭 → 승인/반려(반려는 코멘트 필수) → 요청자 알림.
- **부서 업무함 (M-15-3)**: 완료 처리 → 지연 경고 해제. `처리 Form 열기` = UI Designer 폼.
- **Project Folder·이력 (M-15-8/9)**: 폴더 5종·DXF 더블클릭=CAD 뷰어·sys_history diff.

## 요구사항 접수 (개발서버 전용)

타이틀바 **📝** → 등록 탭: 제목·분류(수정/버그/기능)·우선순위·내용 + **스크린샷**(파일 선택 또는 Ctrl+V 붙여넣기). 현재 화면이 자동 첨부됩니다. 목록 탭에서 처리 상태(접수/처리중/완료/반려) 확인 — 개발 라운드에서 일괄 반영됩니다.

## 자주 겪는 상황

| 증상 | 원인·해결 |
|---|---|
| "백엔드 연결 필요" 로 저장 거부 | MOCK 모드(서버 불가) — 잠시 후 재시도, 지속 시 관리자 문의 |

| 도면/부품 삭제 409 | 발행(RELEASED)·BOM 참조 등 보호 규칙 — 참조 해제 후 삭제 |
