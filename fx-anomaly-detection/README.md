## 문의 개요

은행 내 AI-FDS 는 이미 운영 중이나, **외환업무(송금·수출·수입)에 특화된 AI 이상거래 탐지체계**를 별도 구축하는 방안을 검토 중. 구상 중인 구축 방향(첨부 자료 기준):

1. 거래 및 이상거래 데이터 확보
2. AI 탐지 솔루션 연계
3. 실시간 탐지 및 업무 적용
4. 운영체계 및 지속 개선

## 요청 사항 (유사 구축 사례 기준 대략적 범위)

- 구축 단계별 예상 기간
  - 데이터 준비
  - 환경 준비(솔루션 연계)
  - 시범운영(Pilot)
  - 실운영 전환
- 단계별 예상 투입 인원 (PM, AI, 데이터, 개발 등)
- 전체 구축 비용의 대략적인 범위
- 구축 시 주요 고려사항 및 선행 준비사항
- 공개 가능한 구축 사례 또는 참고자료
