# PM 사전영업 조사 보고서 — AI 외환 이상거래 탐지체계 구축

작성일: 2026-07-24 · 상태: 내부용

## 1. 사업 배경 — 왜 지금 이 문의가 나왔는가

### 1.1 이상 외환송금 사태 (2022~)

- 전 은행권에서 이상 외화송금이 확인되었고, 금감원 파악 기준 **53.7억 달러(약 7조 원), 44개 업체** 관여. 자체검사 포함 시 8.5조 원 이상으로 보도됨.
- 은행별 송금 규모: 신한 23.6억 달러, 우리 16.2억, 하나 10.8억, 국민 7.5억, 농협 6.4억 달러.
- 전형 구조: **가상자산거래소 출금 → 무역법인(귀금속·여행업 등) 계좌 → 홍콩·일본·미국·중국 해외법인 송금** — 김치 프리미엄 차익거래 의심. 수출입 송장을 가장한 송금이라 기존 Rule 로 걸러지지 않음.
- 결과: 검찰·세관 합동 수사, 금감원 제재심(과징금·업무정지 검토), 은행 외환 **내부통제 강화 압박**이 상시화. 이후에도 가상자산 관련 외환거래 규제 회피가 반복되어 금감원이 은행 외환 담당 소집·경고를 지속.

→ **시사점**: 이 문의는 일회성 아이디어가 아니라 은행권 공통의 규제·감독 압력에서 나온 수요. 첨부 슬라이드의 데이터 소스(해외이상송금점검내역, 외화송금제어등록 등)는 사태 이후 만들어진 점검 체계의 산출물로 보이며, 그 Rule 체계를 AI로 고도화하려는 단계임.

### 1.2 TBML(무역기반 자금세탁) 규제 강화

- FATF 중심으로 TBML 국제 규제 기준이 지속 강화, 미국발 AML 제재가 아시아로 확산.
- 문의 본문이 송금뿐 아니라 **수출·수입**을 명시한 것은 TBML 대응(수출입 서류·물품 대비 대금 흐름 점검)까지 시야에 둔 것으로 해석됨.

## 4. 리스크 / 기회 요인

**리스크**
1. 사전 검토 단계 정보 제공이 그대로 RFP 스펙이 되고, 발주는 경쟁입찰로 갈 수 있음 — 회신은 성실하되 방법론 상세(모델 설계, Feature 셋 등)는 미팅 단계로 유보.
2. 개인 gmail — 소속·권한 미확인 상태. 비용 하한을 너무 낮게 제시하면 이후 협상 기준점이 됨.
3. 금융권 실적(레퍼런스) 요구 가능성 — 자사 공개 가능 실적 목록 확정 필요 (사용자 결정 대기).
4. 차세대 일정 종속 — 수주해도 착수가 밀릴 수 있음. 데이터 준비 단계만 선행 분리 제안으로 완화 가능.

**기회**
1. 규제 압력 기반 수요라 사업 자체가 사라질 가능성은 낮음 (연기는 가능).
2. 데이터 준비(라벨 정비·Feature 설계) 선행 컨설팅을 소규모 선행 사업으로 분리 제안하면 본 사업 전 관계 선점 가능.
3. 문의 범위가 송금→수출입 확장 구조라 다년 단계형 사업으로 키울 여지.

## 5. 문의자 확인 질문 리스트 (회신 말미 또는 미팅용)

1. 적용 대상 채널·업무 범위: 1차 해외송금 한정인지, 수출입(TBML)까지 동시 착수인지
2. 이상거래 판정(Label) 데이터: 축적 기간, 건수 규모, 판정 기준의 일관성 여부
3. AI 탐지 솔루션: 상용 도입 전제인지, 자체(협력) 개발 포함 검토인지
4. 차세대 일정: 데이터 구조 확정 시점 — 본 사업 착수 가능 시점 역산용
5. 기존 AI-FDS 와의 관계: 별도 시스템인지, 케이스 관리·알림 통합 전제인지
6. 인프라 제약: 망분리 환경, 학습 환경(온프레미스/클라우드) 정책
7. 목표 성능 지표: 오탐률·검토 가용량 등 운영 목표 수준
