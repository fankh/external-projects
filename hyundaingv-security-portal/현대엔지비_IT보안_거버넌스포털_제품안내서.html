I. 시스템 개요 — 전사 IT·보안 통합 포털

IT 부서의 투자·비용, 인프라 운영, 개발 요청(SR), 프로젝트, 그리고 전사 보안 컴플라이언스 업무를 단일 포털에서 처리하는 통합 관리 시스템이다. 모든 업무는 공통 전자결재와 메뉴·기능 권한관리 체계 위에서 운영된다.

IT 운영 거버넌스

  • IT 투자/비용 계획·집행
  • SR(개발/데이터/권한) 관리
  • 인프라·자산·장애 관리
  • 프로젝트 진행 관리

보안 거버넌스

  • 보안서약서 전사 징구
  • 재택근무·출력물 관리
  • 보안위반 관리
  • 보안교육·점검(ISMS)

공통 기반

  • 전자결재 워크플로우
  • 메뉴·기능 권한관리
  • 공통코드·엑셀양식
  • 사내 시스템 연동
설계 원칙 — 단일 통제 평면: 개별 시스템에 흩어진 IT·보안 업무를 하나의 포털로 통합하여, 전 업무에 동일한 결재·권한·감사 기준을 일관되게 적용한다.

IV. 보안 거버넌스 — 임직원 보안의식 제고

보안서약서 (전사 징구)

  • 일반·관리책임자·재택근무·특별(보안담당자) 서약서
  • 연도별 양식(개정일자) 관리, 동의 진행·이력
  • 협력업체 서약서 — 징구·첨부·결재(company_pledges)
  • 부서서약현황 — 부서·전사 진행현황, 미서약 안내메일(입사·복직·개정 기준)
  • Knox 계정 미보유자 스캔본 업로드

재택 · 출력물 · 보안위반

  • 재택근무 체크리스트 — 대상자 관리, 주기별 동의·조회
  • 출력물 개인정보관리 — 폐기현황 일배치 이관, 본인·부서 결재상신(printout_discards)

근태·인사 연동: 서약 대상·미서약자 판별은 HR-HUB/PTMS의 인사·근태 정보와 연동되며, 안내메일이 자동 발송된다.

IV. 보안 거버넌스 — 보안 컴플라이언스 (교육·점검)

보안교육

  • 연간계획 — 결재완료 계획 업로드·조회(연도별)
  • 결과관리 — 과정별 교육결과·명단 등록
  • 이수현황 — 결과 기반 이수현황 조회
  • 게시/공지 이력 — 완료 내역 업로드·조회

보안점검 (ISMS)

  • 기준관리 — 점검 항목 템플릿, ISMS·외부기관 항목 코드 관리(대·중분류)
