# Breach Response Runbook (GDPR Art.33)
## When: Prometheus alert `BreachAuthorityDeadlineLate` or manual report
1. Go to `/admin/compliance` → Breach Register
2. Verify breach details: severity, affected records, data categories
3. If not yet filed: File New Breach with all known details
4. 72-hour clock starts from detection time
5. Notify DPA: click "Notify DPA" button, enter reference number
6. If high-risk to subjects: click "Notify Subjects" with count
7. Document containment actions in the breach record
8. Close when investigation + remediation complete
