목적 (Purpose)

본 기능확인서는 AI 기반 2D 도면 생성 시스템(ian-ai-blueprint)의 구축된 시스템이 요구사항에 맞게 구현되었는지를 확인하기 위한 문서입니다. 고객사와 수행사가 함께 진행하는 기능 확인 프로세스의 결과를 기록합니다.

범위 (Scope)

요구사항 정의서의 기능 요구사항 FR-001~FR-006(CAD 파일 업로드, 2D 도면 렌더링, AI 도면 생성, DXF 내보내기, 레이어 관리, IFC 층 선택)과 비기능 요구사항(50MB 업로드 제한, DWG 변환기 미설정 시 501 안내 등)에 대한 기능 확인을 범위로 합니다.
